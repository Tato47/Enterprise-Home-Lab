# Network Diagram

## Overview

This network diagram represents the architecture of the Enterprise Home Lab environment.

## Systems Included

- Router / Internet Gateway
- Windows Server 2022
- Active Directory Domain Services (AD DS)
- DNS Server
- Windows 11 Client
- Kali Linux Workstation

## Purpose

This configuration simulates a small enterprise network for:

- Active Directory administration
- Windows Server management
- DNS and DHCP configuration
- Domain joining and authentication
- Security testing from Kali Linux
- Cybersecurity lab exercises
